## Project Description
A fully secured relational database for an online 
shopping platform built using MySQL. The project 
covers schema design, data population, SHA-256 
password hashing, and Role-Based Access Control (RBAC).

## Security Features Implemented
- SHA-256 cryptographic password hashing
- Role-Based Access Control (GRANT & REVOKE)
- Admin role: ALL PRIVILEGES
- Customer role: SELECT and INSERT only
- All 3 security tests except RBAC which i tested using MYSQL workbench passed via MySQL CLI
